# 功能介绍
## 简介

    项目采用MVVM架构开发，Dotnet版本为8.0。
    Avalonia 版本探索版；

### 主要使用到的第三方库如下
    * CommunityToolkit.Mvvm
    * HandyControl
    * NLog
    * RestSharp
    * WebDav.Client
    * Newtonsoft.Json
    * AvalonEdit

CutStr(string, int)

裁切字符串（中文按照两个字符计算，裁切前会先过滤 Html 标签）

public static string CutStr(this string str, int len)

Parameters

str string

旧字符串

len int

新字符串长度

Returns

string

Remarks

注意：

  1. 若字符串被截断则会在末尾追加“...”，反之则直接返回原始字符串。
  2. 中文按照两个字符计算。若指定长度位置恰好只获取半个中文字符，则会将其补全，如下面的例子： 
    string str = "感谢使用uoLib模块。";
string A = CutStr(str,4);   // A = "感谢..."
string B = CutStr(str,5);   // B = "感谢使..."

CutStr(string, int, bool)

裁切字符串（中文按照两个字符计算）

public static string CutStr(this string str, int len, bool HtmlEnable)

Parameters

str string

旧字符串

len int

新字符串长度

HtmlEnable bool

为 false 时过滤 Html 标签后再进行裁切，反之则保留 Html 标签。

Returns

string

Remarks

注意：

  1. 若字符串被截断则会在末尾追加“...”，反之则直接返回原始字符串。
  2. 参数 HtmlEnable 为 false 时会先调用 uoLib.Common.Functions.HtmlFilter 过滤掉 Html 标签再进行裁切。
  3. 中文按照两个字符计算。若指定长度位置恰好只获取半个中文字符，则会将其补全，如下面的例子： 
    string str = "感谢使用uoLib。";
string A = CutStr(str,4);   // A = "感谢..."
string B = CutStr(str,5);   // B = "感谢使..."
